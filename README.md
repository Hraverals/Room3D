본 리포지토리는 **Room3D** 서비스의 초기 개발 버전이자 전신이 되는 프로젝트입니다.<br>
기능 고도화 및 아키텍처 통합이 완료된 **최종 프로젝트**는 아래 링크에서 확인하실 수 있습니다.<br> 최종 버전에서 더 완성도 높은 코드를 사용해보세요!!

👉 **[Room3D 최종 리포지토리](https://github.com/Hraverals/room-3d)**


## 프로젝트 개요 (Overview)
이곳은 **Room3D**의 핵심 아이디어인 '2D 이미지의 3D 오브젝트 변환' 가능성을 검증하고, 초기 클라이언트-서버 통신 구조를 설계한 공간입니다..

현재 배포된 서비스의 기술적 기반이 되었으며, 다음과 같은 초기 개발이 진행되었습니다.

* 이미지 기반 3D 생성 모델의 웹 서비스 적용 가능성 테스트
* 클라이언트와 서버 간의 기본 데이터 전송 로직 구현
* 초기 사용자 인터페이스 설계 및 테스트

## 개발 히스토리 (History)
이 프로젝트는 이후 [glb_viewer_test 레포지토리](https://github.com/Hraverals/glb_viewer_test)의 개발 단계를 거쳐, 현재의 컨테이너화 작업(Docker) 및 허깅페이스 Quota 의존성 탈피를 위한 클라우드 GPU(Modal)가 적용된 최종 버전으로 발전하였습니다.
